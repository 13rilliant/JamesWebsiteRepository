Link #2 Page

Link to index

Coded as:

<a href="../index.html">Link to index</a>

On our Index page, we looked at links which are targeting files beneath the folder which the given file is located in. In understanding that concept, it becomes equally valuable to understand how to link to files above the folder which a given page is currently in.

We arrived at this page from "index.html" by targeting this page in a sub-folder from where "index.html" was located. Now that we're on this file, we consider this our current page and target other links in relation to it.

To select a folder one level up, use "../" followed by the name of the file you want to reach in that one tier up. An example of this would be: "../made_up_file_name". And if you want to seek up multiple folders, simply repeat this inline. Navigating up four folders would look like this: "../../../../". Now, since we are currently on "link2.html", we need still want to target "link1.html". We achieve this by telling the server the following:

<a href="../index.html">Link to index</a>

Now the server knows to look one folder higher and then try to find a file called, "index.html".
